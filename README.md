# Subscription App (Spring Boot + JSP)

## Requirements
- Java 17
- PostgreSQL

## Database
Create a database (e.g. `subscriptiondb`) and run:
- `src/main/resources/db/init.sql`

## Run
```bash
mvn spring-boot:run
```

Open:
- http://localhost:8080/

## Env vars (optional)
- DB_URL
- DB_USER
- DB_PASSWORD
